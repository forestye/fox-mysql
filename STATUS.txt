💡 支持的参数类型（完整）
========================================

现在完全支持：

✅ 基础类型:
  - int id = 123; conn->query_prepared("...", id);
  - double price = 9.99; conn->query_prepared("...", price);
  - long long big = 123456789; conn->query_prepared("...", big);

✅ 字符串类型:
  - std::string name = "Alice"; conn->query_prepared("...", name);  // 左值
  - conn->query_prepared("...", "Alice");                           // 字面量
  - conn->query_prepared("...", std::string("Alice"));             // 临时对象
  - const char* str = "test"; conn->query_prepared("...", str);
  - std::string_view sv = "view"; conn->query_prepared("...", sv);

✅ NULL值:
  - conn->query_prepared("...", nullptr);

✅ 混合使用:
  - std::string name = "Alice"; int age = 25;
  - conn->query_prepared("INSERT INTO t VALUES (?, ?)", name, age);

✅ 循环中使用:
  - for (const auto& item : items) {
      conn->execute_prepared("INSERT ...", item.name, item.id);
    }
